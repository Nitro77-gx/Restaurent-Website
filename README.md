# Reestar Restaurant Website

A simple restaurant website built using HTML and CSS.

## Features
- Clean navigation bar
- Food menu cards with images
- Dish descriptions
- Order Now buttons
- Contact form section
- Responsive and beginner-friendly design

## Technologies Used
- HTML5
- CSS3

## Project Preview
The website displays various food items along with descriptions and provides a contact section for users to get in touch.
